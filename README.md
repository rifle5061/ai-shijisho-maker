# AI指示書メーカー v2

AIに貼れるプロンプトを、入力フォームから作成する静的サイトです。

## 変更点

- 未入力項目はプロンプトに反映しない
- PR表記を選択式に変更
- スマホでEnterを押すと次項目へ移動
- AI API不要

## 使い方

1. `index.html`
2. `style.css`
3. `script.js`

をGitHubリポジトリ直下に置いて、GitHub Pagesを有効化してください。


## v3 変更点

- PR表記の選択欄を一番下に移動
- 項目番号を調整


## v4 変更点

- 未入力項目を確実に出力しないよう script.js を全面修正
- index.html の script 読み込みに `?v=4` を付けてキャッシュ対策
- フッターに v4 表示を追加
